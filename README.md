- **Create Recipes**: Easily create and share your favorite recipes with the community.

- **CRUD Operations**: Implement Create, Read, Update, and Delete operations to manage your recipe collection effectively.

## How to run
 1. create .env:
```

DISCORD_CLIENT_ID=""

UPLOADTHING_APP_ID=

SHADOW_DATABASE_URL=

```
 2. run dev :
```bash
npm run db:push
npm run dev
```
